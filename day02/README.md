## sizeof
|    type     | size |
| :---------- | ---: |
| int         |  4   |
| short int   |  2   |
| long int    |  8   |
| unsigned int|  4   |
| char        |  1   |
| float       |  4   |
| double      |  8   |
| long double |  16  |

## printf
> * %lu 为 32 位无符号整数
> * %E 为以指数形式输出单、双精度实数

## 二进制表示
> * float 	单精度浮点值。单精度是这样的格式，1位符号，8位指数，23位小数。
> * double 	双精度浮点值。双精度是1位符号，11位指数，52位小数。
